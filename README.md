# 🚗 AI Motoriste

سيارة ذكية ذاتية القيادة مبنية بالذكاء الاصطناعي

## المطور
**يوسف** - طالب من موريتانيا عمره 14 سنة

## ما يفعله المشروع
- 🧠 تتعلم من التجربة باستخدام Q-Learning
- 🗺️ تجد أقصر طريق باستخدام A*
- 🔴 تتجنب الحواجز المتحركة تلقائياً
- ✅ تصل للهدف في مدينة 10x10

## التقنيات المستخدمة
- Python
- NumPy
- خوارزمية A*
- Q-Learning

## كيف تشغله
```bash
pip install numpy
python car_ai.py
```
